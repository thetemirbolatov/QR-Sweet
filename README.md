QR Sweet v1.0.0 
---


## Обзор

QR Sweet — это десктопное приложение для Windows, которое позволяет создавать классические QR-коды и расшифровывать их из изображений. Программа поддерживает перетаскивание файлов, автоматическую генерацию при вводе текста и сохранение результата в формате PNG.

Приложение распространяется в виде готового установщика. Исходный код в репозитории не публикуется.

---

## Возможности

### Создание QR-кода
- Ввод произвольного текста или URL-адреса
- Автоматическая генерация QR-кода при вводе данных
- Классический черно-белый QR-код
- Сохранение в файл формата PNG

### Расшифровка QR-кода
- Перетаскивание изображения в окно программы (drag-and-drop)
- Выбор файла через стандартный диалог
- Поддержка форматов: PNG, JPG, JPEG, BMP, TIFF, WEBP
- Автоматическое распознавание с использованием OpenCV
- Отображение расшифрованного содержимого в текстовом поле

### Интерфейс
- Фиксированный размер окна: 900 на 650 пикселей
- Две панели: создание и расшифровка
- Классический светлый дизайн
- Поддержка отображения на русском языке

## Установка

### Способ 1: Установщик (рекомендуется)

1. Перейдите в раздел [Releases](https://github.com/thetemirbolatov/qr-sweet/releases)
2. Скачайте файл `QR_Sweet_Setup.exe`
3. Запустите установщик от имени администратора
4. Следуйте инструкциям мастера установки
5. После завершения установки программа будет доступна в меню «Пуск» и на рабочем столе (опционально)

### Способ 2: Портативная версия

1. Перейдите в раздел [Releases](https://github.com/thetemirbolatov/qr-sweet/releases)
2. Скачайте архив `QRSweet_Portable.zip`
3. Распакуйте архив в любую папку
4. Запустите `QRSweet.exe`

## Использование

### Создание QR-кода

1. Запустите программу
2. В левой панели «Создать QR-код» введите текст или URL-адрес в поле ввода
3. QR-код будет сгенерирован автоматически и отобразится в области предпросмотра
4. Для сохранения нажмите кнопку «Сохранить как PNG»
5. Выберите место сохранения и имя файла

### Расшифровка QR-кода

1. Запустите программу
2. Перетащите изображение с QR-кодом в правую панель «Расшифровать QR-код»
   ИЛИ нажмите кнопку «Выбрать файл с QR-кодом» и укажите путь к изображению
3. Расшифрованный текст отобразится в поле «Результат»

## Решение проблем

### Программа не запускается

- Убедитесь, что у вас установлены актуальные версии Visual C++ Redistributable
- Запустите программу от имени администратора
- Проверьте, не блокирует ли антивирус запуск

### QR-код не распознается

- Используйте изображения высокого качества без размытия
- Убедитесь, что QR-код полностью помещается в кадр
- Проверьте освещение: QR-код должен быть равномерно освещен

### Ошибка при сохранении

- Проверьте права доступа к выбранной папке
- Убедитесь, что на диске достаточно свободного места
